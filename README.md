# Directory Analyzer

A simple and efficient command-line tool built using **Rust** to analyze directories and understand file system usage. It scans folders recursively and provides useful information about files, folders, and storage.

##  About the Project

Managing large folders can become difficult when there are hundreds of files and subfolders. Directory Analyzer helps users quickly explore a directory and get a summary of its contents.

This project was built to learn and apply Rust concepts like file system handling, recursion, and efficient memory management.

##  Features

* Analyze folders and subfolders recursively
* Count total files and directories
* Calculate total storage usage
* Identify file types and extensions
* Fast and lightweight command-line execution

##  Technologies Used

* **Rust**
* **Cargo**
* Rust Standard Library (`std::fs`, `std::path`)

##  How to Run

### Clone the repository

```bash
git clone https://github.com/InumulaTeja/Directory-Analyzer.git
```

### Navigate to the project folder

```bash
cd Directory-Analyzer
```

### Run the application

```bash
cargo run
```

Enter the directory path when prompted, and the tool will analyze the folder.
